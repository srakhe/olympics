Olympics Project

This is an extensive application that provides in-depth analysis and predictions related to the Olympics.

The following are the key functionalities of this application:

1. Get people's views on particular olympic games that were hosted. Get Views

2. View the effects that hosting olympics has had on the various economic factors in that country. View Effects

3. View the trends that the number of medals won by each country have had when the countries have hosted the olympics. View Medal Trends

4. Get predictions on the various economic factors if a country is selected to host the olympics in a particular year. Get Predictions
